# Paccy To Do List App

A futuristic, AI-powered To-Do List and Student Planner application.

## Features
- **Statistical Dashboard**: Visual analytics of your task completion performance.
- **Student Integration**: Manage courses and link tasks to specific subjects.
- **Smart Planning**: "Paccy AI Helper" (Simulated) to help draft study schedules.
- **Calendar View**: Weekly schedule visualization with Google Calendar Sync (Mock).
- **Cloud Sync**: Simulated cloud storage with automatic local persistence.
- **Futuristic UI**: Glassmorphism design with Neon aesthetics and Dark/Light mode.

## Technology
- **Core**: HTML5, CSS3, Vanilla JavaScript (ES Modules).
- **Design System**: Custom Glassmorphism CSS variables.
- **Icons**: FontAwesome 6.
- **Fonts**: 'Outfit' and 'Space Grotesk' from Google Fonts.
